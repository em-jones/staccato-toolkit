# OQA Profiles

User and team profiles for OQA.

## Alignment with OQA Ecosystem Spec

Provides identity and preferences for the OQA ecosystem.

### Profile Types

- **User Profile**: Individual user preferences and settings
- **Team Profile**: Group-based configuration for shared dashboards
- **Service Profile**: Service-level metadata for signal attribution

### Features

- Alert notification preferences
- Dashboard customization
- API key management for OQA backends
- Compliance framework access control
