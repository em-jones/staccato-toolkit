---
title: Timeline
description: Vertical timeline components
---

## Usage

Apply `.timeline` with `.timeline-item`, `.timeline-dot`, and `.timeline-box`.

```html
<div class="timeline">
  <div class="timeline-item">
    <div class="timeline-dot"></div>
    <div class="timeline-box">
      <div class="timeline-title">Project Kickoff</div>
      <div class="timeline-date">Jan 2024</div>
      <div class="timeline-text">Initial planning and team formation</div>
    </div>
  </div>
  <div class="timeline-item">
    <div class="timeline-dot"></div>
    <div class="timeline-box">
      <div class="timeline-title">Beta Launch</div>
      <div class="timeline-date">Mar 2024</div>
      <div class="timeline-text">Public beta released</div>
    </div>
  </div>
  <div class="timeline-item">
    <div class="timeline-dot"></div>
    <div class="timeline-box">
      <div class="timeline-title">v1.0 Release</div>
      <div class="timeline-date">Jun 2024</div>
      <div class="timeline-text">Stable release for production</div>
    </div>
  </div>
</div>
```

## Item positions

| Class                  | Description      |
| ---------------------- | ---------------- |
| `.timeline-item-start` | Content on left  |
| `.timeline-item-end`   | Content on right |

## Dot variants

| Class            | Description       |
| ---------------- | ----------------- |
| `.timeline-dot`  | Simple dot        |
| `.timeline-ring` | Ring/bordered dot |
