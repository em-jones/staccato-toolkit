## Usage

Apply `.link` to anchor elements.

```html
<a href="#" class="link">Default link</a>
```

## Color variants

| Class             | Description        |
| ----------------- | ------------------ |
| `.link-primary`   | Primary color      |
| `.link-secondary` | Secondary color    |
| `.link-accent`    | Accent color       |
| `.link-neutral`   | Neutral/text color |

```html
<a href="#" class="link link-primary">Primary</a>
<a href="#" class="link link-secondary">Secondary</a>
<a href="#" class="link link-accent">Accent</a>
<a href="#" class="link link-neutral">Neutral</a>
```

## Style modifiers

| Class          | Description             |
| -------------- | ----------------------- |
| `.link-hover`  | Underline on hover only |
| `.link-active` | Bold text               |
| `.link-ghost`  | Background on hover     |

```html
<a href="#" class="link link-primary link-hover">Hover underline</a>
<a href="#" class="link link-primary link-active">Active link</a>
<a href="#" class="link link-ghost">Ghost link</a>
```
