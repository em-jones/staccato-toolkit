## Usage

Use `.tabs` with `.tabs-list` for tab buttons and `.tabs-content` for the panel. Add `.tab-active` to the active tab.

```html
<div class="tabs">
  <div class="tabs-list">
    <button class="tab tab-active">Tab 1</button>
    <button class="tab">Tab 2</button>
    <button class="tab">Tab 3</button>
  </div>
  <div class="tabs-content">
    <p>Content for the active tab panel</p>
  </div>
</div>
```

## Radio-based tabs (CSS-only toggle)

For CSS-only tab toggling without JavaScript, use radio inputs. The content following a checked radio will be shown.

```html
<div class="tabs tabs-box">
  <input type="radio" name="my_tabs" class="tab" aria-label="Tab 1" />
  <div class="tab-content">Content for Tab 1</div>

  <input type="radio" name="my_tabs" class="tab" aria-label="Tab 2" checked="checked" />
  <div class="tab-content">Content for Tab 2</div>

  <input type="radio" name="my_tabs" class="tab" aria-label="Tab 3" />
  <div class="tab-content">Content for Tab 3</div>
</div>
```

Note: Each tab group needs a unique `name` attribute for the radio inputs.

## Outlined content area

```html
<div class="tabs">
  <div class="tabs-list">
    <button class="tab tab-active">Files</button>
    <button class="tab">Settings</button>
  </div>
  <div class="tabs-content tabs-content-outlined">
    <p>Content with an outlined border below the tabs</p>
  </div>
</div>
```

## Sizes

Five sizes are available: `xs`, `sm`, `md` (default), `lg`, `xl`.

```html
<div class="tabs tabs-xs">
  <div class="tabs-list">
    <button class="tab tab-active">XS Tab</button>
    <button class="tab">Other</button>
  </div>
</div>
<div class="tabs tabs-lg">
  <div class="tabs-list">
    <button class="tab tab-active">LG Tab</button>
    <button class="tab">Other</button>
  </div>
</div>
```

## Layout options

| Class                 | Description        |
| --------------------- | ------------------ |
| `.tabs-list-centered` | Center the tabs    |
| `.tabs-list-full`     | Equal-width tabs   |
| `.tabs-bottom`        | Tabs below content |
