## Usage

Apply `.footer` to create a footer section.

```html
<footer class="footer">
  <div class="footer-section">
    <div>
      <div class="footer-title">Company</div>
      <ul class="footer-list">
        <li><a href="#" class="footer-item">About</a></li>
        <li><a href="#" class="footer-item">Careers</a></li>
      </ul>
    </div>
    <div>
      <div class="footer-title">Support</div>
      <ul class="footer-list">
        <li><a href="#" class="footer-item">Help Center</a></li>
        <li><a href="#" class="footer-item">Contact</a></li>
      </ul>
    </div>
  </div>
  <div class="footer-bottom">
    <p>&copy; 2024 Company</p>
    <div class="footer-social">
      <a href="#" class="footer-item">Twitter</a>
      <a href="#" class="footer-item">GitHub</a>
    </div>
  </div>
</footer>
```

## Alignment

| Class            | Description            |
| ---------------- | ---------------------- |
| `.footer-center` | Centered content       |
| `.footer-start`  | Left-aligned (default) |
| `.footer-end`    | Right-aligned          |

## Horizontal layout

```html
<footer class="footer footer-horizontal">
  <div class="footer-section">
    <span class="footer-title">Brand</span>
  </div>
  <div class="footer-bottom">
    <p>&copy; 2024</p>
  </div>
</footer>
```
