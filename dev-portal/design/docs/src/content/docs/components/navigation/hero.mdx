## Usage

Apply `.hero` to create a banner section with `.hero-content`, `.hero-title`, and `.hero-subtitle`.

```html
<div class="hero hero-mini hero-center">
  <div class="hero-content">
    <h1 class="hero-title">Welcome</h1>
    <p class="hero-subtitle">Subtitle text describing your product</p>
    <button class="btn btn-primary">Get Started</button>
  </div>
</div>
```

## Alignment

| Class          | Description      |
| -------------- | ---------------- |
| `.hero-center` | Centered content |
| `.hero-start`  | Left-aligned     |
| `.hero-end`    | Right-aligned    |

```html
<div class="hero hero-mini hero-start">
  <div class="hero-content">
    <h2 class="hero-title">Left Aligned</h2>
    <p class="hero-subtitle">Starts from the left</p>
  </div>
</div>
```

## Sizes

| Class        | Description          |
| ------------ | -------------------- |
| `.hero-mini` | 30vh minimum height  |
| `.hero-md`   | 50vh minimum height  |
| `.hero-lg`   | 70vh minimum height  |
| `.hero-xl`   | 90vh minimum height  |
| `.hero-full` | 100vh minimum height |
