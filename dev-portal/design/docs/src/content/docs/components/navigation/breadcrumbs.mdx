---
title: Breadcrumbs
description: Show navigation hierarchy
---

## Usage

Apply `.breadcrumbs` to a `<nav>` element to display hierarchical navigation.

```html
<nav class="breadcrumbs">
  <li><a href="/">Home</a></li>
  <li><a href="/products">Products</a></li>
  <li>Current Page</li>
</nav>
```

## Sizes

| Class             | Description           |
| ----------------- | --------------------- |
| `.breadcrumbs-xs` | Extra small text      |
| `.breadcrumbs-sm` | Small text            |
| `.breadcrumbs-md` | Medium text (default) |
| `.breadcrumbs-lg` | Large text            |

```html
<nav class="breadcrumbs breadcrumbs-xs">
  <li><a href="#">Home</a></li>
  <li>XS Page</li>
</nav>
<nav class="breadcrumbs breadcrumbs-sm">
  <li><a href="#">Home</a></li>
  <li>SM Page</li>
</nav>
<nav class="breadcrumbs breadcrumbs-md">
  <li><a href="#">Home</a></li>
  <li>MD Page</li>
</nav>
<nav class="breadcrumbs breadcrumbs-lg">
  <li><a href="#">Home</a></li>
  <li>LG Page</li>
</nav>
```
