---
title: Divider
description: Visual separators for content
---

## Usage

Apply `.divider` to create horizontal or vertical separators.

```html
<div class="divider"></div>
```

## Orientation

| Class               | Description      |
| ------------------- | ---------------- |
| `.divider-vertical` | Vertical divider |

### Orientation Examples

```html
<div class="divider"></div>

<div style="height: 100px">
  <div class="divider-vertical"></div>
</div>
```

## Alignment

| Class             | Description              |
| ----------------- | ------------------------ |
| `.divider-start`  | Content aligned to start |
| `.divider-center` | Content centered         |
| `.divider-end`    | Content aligned to end   |

## With content

Use `.divider-with-content` and `.divider-content` to add text in the middle:

```html
<div class="divider divider-with-content">
  <span class="divider-content">Or continue with</span>
</div>
```

## Colors

| Class                | Description     |
| -------------------- | --------------- |
| `.divider-neutral`   | Neutral color   |
| `.divider-primary`   | Primary color   |
| `.divider-secondary` | Secondary color |
| `.divider-accent`    | Accent color    |
| `.divider-info`      | Info color      |
| `.divider-success`   | Success color   |
| `.divider-warning`   | Warning color   |
| `.divider-error`     | Error color     |

### Color Examples

```html
<div class="divider divider-primary"></div>
<div class="divider divider-success"></div>
<div class="divider divider-warning"></div>
```
