---
title: Loading
description: Loading spinners and indicators
---

## Usage

Apply `.loading` with a spinner variant.

```html
<div class="loading">
  <div class="loading-spinner"></div>
</div>
```

## Spinner types

| Class               | Description             |
| ------------------- | ----------------------- |
| `.loading-spinner`  | Rotating border spinner |
| `.loading-dots`     | Bouncing dots           |
| `.loading-ring`     | Double ring spinner     |
| `.loading-bars`     | Animated bars           |
| `.loading-infinity` | Infinity loop animation |

### Spinner Examples

```html
<div class="loading"><div class="loading-spinner"></div></div>
<div class="loading">
  <div class="loading-dots"><span></span><span></span><span></span></div>
</div>
<div class="loading"><div class="loading-ring"></div></div>
<div class="loading">
  <div class="loading-bars"><span></span><span></span><span></span><span></span></div>
</div>
<div class="loading"><div class="loading-infinity"></div></div>
```

## Sizes

| Class         | Size             |
| ------------- | ---------------- |
| `.loading-xs` | Extra small      |
| `.loading-sm` | Small            |
| `.loading-md` | Medium (default) |
| `.loading-lg` | Large            |
| `.loading-xl` | Extra large      |

### Size Examples

```html
<div class="loading loading-xs"><div class="loading-spinner"></div></div>
<div class="loading loading-sm"><div class="loading-spinner"></div></div>
<div class="loading loading-lg"><div class="loading-spinner"></div></div>
```
