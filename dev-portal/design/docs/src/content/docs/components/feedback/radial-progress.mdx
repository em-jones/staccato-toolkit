---
title: Radial Progress
description: Circular progress indicators
---

## Usage

Apply `.radial-progress` to a `<div>` and set `--value` (0–100) via inline style.

```html
<div class="radial-progress" style="--value: 45">45%</div>
```

## Sizes

| Class                 | Size           |
| --------------------- | -------------- |
| `.radial-progress-xs` | 2rem           |
| `.radial-progress-sm` | 3rem           |
| `.radial-progress-md` | 5rem (default) |
| `.radial-progress-lg` | 8rem           |
| `.radial-progress-xl` | 10rem          |

```html
<div class="radial-progress radial-progress-xs" style="--value: 60">60%</div>
<div class="radial-progress radial-progress-sm" style="--value: 60">60%</div>
<div class="radial-progress radial-progress-md" style="--value: 60">60%</div>
<div class="radial-progress radial-progress-lg" style="--value: 60">60%</div>
```

## Colors

| Class                        | Description     |
| ---------------------------- | --------------- |
| `.radial-progress-primary`   | Primary color   |
| `.radial-progress-secondary` | Secondary color |
| `.radial-progress-accent`    | Accent color    |
| `.radial-progress-warning`   | Warning color   |
| `.radial-progress-error`     | Error color     |

```html
<div class="radial-progress radial-progress-primary" style="--value: 70">70%</div>
<div class="radial-progress radial-progress-secondary" style="--value: 50">50%</div>
<div class="radial-progress radial-progress-accent" style="--value: 85">85%</div>
<div class="radial-progress radial-progress-warning" style="--value: 30">30%</div>
<div class="radial-progress radial-progress-error" style="--value: 90">90%</div>
```
