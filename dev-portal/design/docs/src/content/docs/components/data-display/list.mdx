## Usage

Apply `.list` with `.list-item` for structured list rows.

```html
<ul class="list">
  <li class="list-item">
    <div class="list-item-content">
      <span class="list-item-title">First Item</span>
      <span class="list-item-subtitle">Supporting text</span>
    </div>
  </li>
  <li class="list-item">
    <div class="list-item-content">
      <span class="list-item-title">Second Item</span>
      <span class="list-item-subtitle">More detail here</span>
    </div>
  </li>
  <li class="list-item">
    <div class="list-item-content">
      <span class="list-item-title">Third Item</span>
    </div>
  </li>
</ul>
```

## Variants

| Class           | Description           |
| --------------- | --------------------- |
| `.list-hover`   | Hover effect on items |
| `.list-divided` | Border between items  |
| `.list-none`    | No hover or dividers  |

```html
<ul class="list list-divided list-hover">
  <li class="list-item">Item 1</li>
  <li class="list-item">Item 2</li>
  <li class="list-item">Item 3</li>
</ul>
```
