---
title: Indicator
description: Floating badges and status dots anchored to a container element
---

## Usage

Wrap the target element with `.indicator` and place `.indicator-item` inside it. By default the
badge anchors to the **top-end** corner.

```html
<div class="indicator">
  <button class="btn">Notifications</button>
  <span class="indicator-item">3</span>
</div>
```

## Position classes

Combine one vertical and one horizontal modifier on `.indicator-item` to control placement.

| Class               | Axis       | Description          |
| ------------------- | ---------- | -------------------- |
| `.indicator-top`    | vertical   | Align to top edge    |
| `.indicator-middle` | vertical   | Center vertically    |
| `.indicator-bottom` | vertical   | Align to bottom edge |
| `.indicator-start`  | horizontal | Align to left edge   |
| `.indicator-center` | horizontal | Center horizontally  |
| `.indicator-end`    | horizontal | Align to right edge  |

## Dot variant

Add `.indicator-dot` to render a small dot instead of a labelled badge. Useful for presence or
status indicators on avatars.

```html
<div class="indicator">
  <div class="avatar avatar-lg">JD</div>
  <span
    class="indicator-item indicator-dot indicator-bottom indicator-end"
    style="background: var(--color-accent); border: 2px solid var(--color-bg)"
  ></span>
</div>
```

### Dot positions
