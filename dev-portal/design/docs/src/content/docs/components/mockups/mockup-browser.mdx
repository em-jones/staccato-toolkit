## Usage

Apply `.mockup-browser` to create a browser window mockup.

```html
<div class="mockup-browser">
  <div class="mockup-browser-toolbar">
    <div class="mockup-browser-dot" style="background: #ff5f56"></div>
    <div class="mockup-browser-dot" style="background: #ffbd2e"></div>
    <div class="mockup-browser-dot" style="background: #27c93f"></div>
    <div class="mockup-browser-address">example.com</div>
  </div>
  <div class="mockup-browser-content" style="padding: 1.5rem; min-height: 6rem;">
    Your content here
  </div>
</div>
```

## Structure

| Class                     | Description                               |
| ------------------------- | ----------------------------------------- |
| `.mockup-browser`         | Main container                            |
| `.mockup-browser-toolbar` | Browser toolbar with dots and address bar |
| `.mockup-browser-address` | Address bar input                         |
| `.mockup-browser-dot`     | Window control dots                       |
| `.mockup-browser-content` | Main content area                         |
