## Usage

Apply `.mockup-window` to create a window mockup.

```html
<div class="mockup-window">
  <div class="mockup-window-toolbar">
    <div class="mockup-window-dot" style="background: #ff5f56"></div>
    <div class="mockup-window-dot" style="background: #ffbd2e"></div>
    <div class="mockup-window-dot" style="background: #27c93f"></div>
    <div class="mockup-window-title">Document</div>
  </div>
  <div class="mockup-window-content" style="padding: 1.5rem; min-height: 6rem;">
    Your content here
  </div>
</div>
```

## Structure

| Class                    | Description           |
| ------------------------ | --------------------- |
| `.mockup-window`         | Main window container |
| `.mockup-window-toolbar` | Window toolbar        |
| `.mockup-window-dot`     | Window control dots   |
| `.mockup-window-title`   | Window title          |
| `.mockup-window-content` | Main content area     |
