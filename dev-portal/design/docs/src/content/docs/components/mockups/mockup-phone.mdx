## Usage

Apply `.mockup-phone` to create a phone mockup.

```html
<div class="mockup-phone">
  <div
    class="mockup-phone-display"
    style="display: flex; align-items: center; justify-content: center; background: var(--color-surface0); color: var(--color-text);"
  >
    App content
  </div>
</div>
```

## Structure

| Class                   | Description             |
| ----------------------- | ----------------------- |
| `.mockup-phone`         | Main phone container    |
| `.mockup-phone-display` | Screen area             |
| `.mockup-phone-camera`  | Camera notch (optional) |
