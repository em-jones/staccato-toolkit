## Usage

Apply `.kbd` to display keyboard keys.

```html
<kbd class="kbd">Ctrl</kbd> <kbd class="kbd">C</kbd>
```

## Sizes

| Class     | Size             |
| --------- | ---------------- |
| `.kbd-xs` | Extra small      |
| `.kbd-sm` | Small            |
| `.kbd-md` | Medium (default) |
| `.kbd-lg` | Large            |
| `.kbd-xl` | Extra large      |

### Size Examples

```html
<kbd class="kbd kbd-xs">Esc</kbd>
<kbd class="kbd kbd-sm">Tab</kbd>
<kbd class="kbd kbd-md">Space</kbd>
<kbd class="kbd kbd-lg">Enter</kbd>
```

## Style modifiers

| Class           | Description                     |
| --------------- | ------------------------------- |
| `.kbd-keyboard` | Deeper shadow for keyboard look |
| `.kbd-dark`     | Dark background                 |
| `.kbd-primary`  | Primary color background        |

### Style Examples

```html
<kbd class="kbd kbd-keyboard">Cmd</kbd>
<kbd class="kbd kbd-dark">Alt</kbd>
<kbd class="kbd kbd-primary">Del</kbd>
```

## Key combinations

```html
<kbd class="kbd">Ctrl</kbd> + <kbd class="kbd">C</kbd>
```
