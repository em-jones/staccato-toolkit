---
title: Join
description: Group elements with joined borders
---

## Usage

Apply `.join` to a container to merge multiple elements with connected borders.

```html
<div class="join">
  <button class="btn join-item">First</button>
  <button class="btn join-item">Second</button>
  <button class="btn join-item">Third</button>
</div>
```

## Vertical join

```html
<div class="join join-vertical">
  <button class="btn join-item">First</button>
  <button class="btn join-item">Second</button>
  <button class="btn join-item">Third</button>
</div>
```

## With form inputs

```html
<div class="join">
  <input class="input join-item" placeholder="Search" />
  <button class="btn btn-primary join-item">Search</button>
</div>
```

## Mixed colors

```html
<div class="join">
  <button class="btn btn-primary join-item">Primary</button>
  <button class="btn btn-secondary join-item">Secondary</button>
  <button class="btn btn-accent join-item">Accent</button>
</div>
```
