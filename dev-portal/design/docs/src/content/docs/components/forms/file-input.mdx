---
title: File Input
description: File upload input styling
---

## Usage

Apply `.file-input` to style a file input element.

```html
<input type="file" class="file-input" />
```

## Sizes

| Class            | Description      |
| ---------------- | ---------------- |
| `.file-input-xs` | Extra small      |
| `.file-input-sm` | Small            |
| `.file-input-md` | Medium (default) |
| `.file-input-lg` | Large            |

```html
<input type="file" class="file-input file-input-xs" />
<input type="file" class="file-input file-input-sm" />
<input type="file" class="file-input file-input-md" />
<input type="file" class="file-input file-input-lg" />
```

## Disabled state

```html
<input type="file" class="file-input" disabled />
```
