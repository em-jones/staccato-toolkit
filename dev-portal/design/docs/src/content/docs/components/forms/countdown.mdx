---
title: Countdown
description: Animated numeric countdown display
---

## Usage

Wrap `<span>` elements in `.countdown`. Set `--value` on each span to the number to display. The
browser renders the number using a CSS counter — no JavaScript needed for static display.

```html
<span class="countdown text-4xl font-bold">
  <span style="--value:12"></span>: <span style="--value:05"></span>:
  <span style="--value:47"></span>
</span>
```

## With labels

Pair with small text labels beneath each unit.

```html
<div class="flex gap-4 text-center">
  <div>
    <span class="countdown text-5xl font-bold"><span style="--value:02"></span></span>
    <div class="text-xs mt-1">days</div>
  </div>
  <div>
    <span class="countdown text-5xl font-bold"><span style="--value:10"></span></span>
    <div class="text-xs mt-1">hours</div>
  </div>
  <div>
    <span class="countdown text-5xl font-bold"><span style="--value:24"></span></span>
    <div class="text-xs mt-1">min</div>
  </div>
  <div>
    <span class="countdown text-5xl font-bold"><span style="--value:59"></span></span>
    <div class="text-xs mt-1">sec</div>
  </div>
</div>
```

## Updating with JavaScript

Use JavaScript to set `--value` on the span elements to update the display:

```js
const spans = document.querySelectorAll(".countdown span");
// Update seconds
spans[2].style.setProperty("--value", remainingSeconds);
```
