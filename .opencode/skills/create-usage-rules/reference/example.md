# Rules for working with Ash Framework

## Understanding Ash Framework

Ash is an opinionated, composable framework for building applications in Elixir.
It provides a declarative approach to modeling your domain with resources at the center.
Read [documentation](./usage-rules/actions.md) before attempting to use its features.
Do not assume that you have prior knowledge of the framework or its conventions.
For specific framework utilities, refer to [specific rules](./usage-rules/actions.md)
