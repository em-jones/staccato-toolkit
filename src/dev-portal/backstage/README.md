# Backstage for Staccato toolkit

## Scaffolder customizations

### Core Templates

- `addGitHooksManager` - adds git hooks manager to the project
- `addGitHook` - adds a git hook to the project

### Dependency Management

#### Tech radar

- `addToTechRadar` - adds a package to the tech radar

#### Devbox

**NOTE**: depends on [tech radar](#tech-radar)

- `newDevboxModule` - add's devbox configuration and listed dependencies
- `editDevboxModule` - updates devbox configuration
